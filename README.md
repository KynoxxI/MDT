# MDT RP — Terminal de Données Mobile

Interface web simulée pour les forces de l’ordre RP (GTA RP, FiveM, etc.)

## Fonctions incluses
- Dashboard
- Fiches citoyens
- Fiches véhicules
- Dossiers judiciaires
- Mandats
- Rapports
- Administration
- Connexion avec rôles (agent, sergent, détective, admin)

## Hébergement
Compatible GitHub Pages.  
Ouvrir `index.html` dans un navigateur ou déployer sur Netlify.
